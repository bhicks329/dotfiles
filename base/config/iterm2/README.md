# iTerm2 Configuration

iTerm2 stores most of its configuration in ~/Library/Preferences/com.apple.iTerm2.plist

To export your iTerm2 preferences:
1. Open iTerm2 → Settings → General → Preferences
2. Check 'Load preferences from a custom folder or URL'
3. Set the folder to: ~/source/dotfiles/base/config/iterm2 (or machine override)

This directory can contain:
- Color schemes (.itermcolors files)
- Dynamic profiles (DynamicProfiles/)
- Scripts (Scripts/)
